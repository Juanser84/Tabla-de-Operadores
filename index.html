Operador Tipo Descripción
== Comparación verdadero si el valor izquierdo es igual al derecho.
!= Comparación verdadero si los valores son diferentes.
> Comparación verdadero si el valor izquierdo es mayor que el derecho.
< Comparación verdadero si el valor izquierdo es menor que el derecho.
>= Comparación verdadero si el valor izquierdo es mayor o igual que el derecho.
<= Comparación verdadero si el valor izquierdo es menor o igual que el derecho.
