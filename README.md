# 📋 Tabla de Operadores - Comparación y Lógicos

## 📝 Descripción

Proyecto básico de HTML y CSS que muestra una tabla con los **operadores de comparación** y **operadores lógicos** más utilizados en programación.  
Desarrollado como actividad para practicar tablas HTML y estilos CSS (bordes, colores, tipografía y espaciado).

## ✅ Requisitos cumplidos

- [x] Tabla con tres columnas: **Operador**, **Tipo** y **Descripción**.
- [x] Contiene los **operadores de comparación**:  
  `=`, `!=`, `>`, `<`, `>=`, `<=`
- [x] Contiene los **operadores lógicos**:  
  `AND`, `OR`, `NOT`
- [x] Estilos CSS aplicados:
  - Bordes en tabla y celdas.
  - Colores de fondo (cabecera azul oscuro, filas alternas claras).
  - Tipografía moderna (`'Segoe UI'`, etc.).
  - Espaciado con `padding` y márgenes.
